\# Security Policy



\## Scope



This repository is an educational reverse-engineering challenge built around a self-contained toy program.



It does not target third-party software, commercial games, anti-cheat systems, DRM, or protected production systems.



\## Reporting Security Issues



If you find a security issue in this repository, please open a GitHub issue with a clear description and reproduction steps.



Do not use this project to attack, modify, or bypass protections in software you do not own or do not have permission to analyze.



\## Responsible Use



This project is intended for educational purposes only.
